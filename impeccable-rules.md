# Impeccable UI Rules

## Typography & Hierarchy
- Use fluid type scaling. Never pair more than two font families.
- Maintain a strict vertical rhythm using a 4px/8px grid base.

## Colors & Contrast
- Never use pure black (#000) or pure white (#fff). Use tinted neutrals.
- Ensure all text passes WCAG AA contrast ratios using OKLCH color space.

## Anti-Gravity Visuals
- Use soft, multi-layered diffused shadows instead of harsh solid borders to create depth.
- Apply `backdrop-filter: blur()` and subtle transparency for glassmorphism effects.
- Elements must feel weightless; use smooth acceleration transitions on scroll and hover.
